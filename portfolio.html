Carbon Nano-Onion (CNO) Structure Generator ( NOT READY FOR USE)

An advanced web-based tool for generating multi-shell carbon nano-onion structures using Goldberg-Coxeter (GC) construction with Fibonacci-guided bond optimization. Features real-time 3D visualization (WebGL + Crystal Toolkit), bond-length histogram analysis (1st/2nd/3rd nearest neighbors), and exports to XYZ, CIF, and LAMMPS formats for molecular dynamics simulations.

Key features include configurable shell parameters (C60, C240, C540, etc.), optional reference structure fitting for α-parameter optimization, interactive 3D preview with per-shell filtering, and comprehensive structural analysis tools for computational materials research.
